# Team Action Item System – How This Works

## Files Created
- daily_action_items_master.md → Cursor-first master list (human + AI readable)

## How You Use This Daily
1. Update this file each morning (5 minutes)
2. Add or adjust action items from meetings
3. Mirror active items into Planner (one plan, buckets = teams)

## Planner Rules
- One Planner plan for all teams
- Buckets = Team names
- Task title format: [Team] – Action
- Due date required

## Automation
- Planner feeds a Power Automate flow
- Flow sends you a 7:00 AM Teams DM
- Message is grouped by team and highlights follow-ups

## Source of Truth
- Cursor file = thinking + drafting
- Planner = execution + notification
