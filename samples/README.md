# Sample Data

Reference files from existing team stand-up workflows. These are used for testing and validating ShowRunner automation.

Binary files (.xlsx, .docx) from the original local setup are not tracked in Git.
They can be found in the SharePoint document library or the original local directory.

## Teams

- **APPS/** — Applications team action items and system documentation
- **GSS/** — Guest & Studio Services daily ledgers and sprint summaries
- **PLAT/** — Platform team daily briefings and Planner imports
